Puttingbana – full Vercel-version

Innehåll:
- index.html
- manifest.json
- sw.js
- vercel.json

Så publicerar du:
1. Packa upp zip-filen.
2. Gå till vercel.com och logga in.
3. Välj Add New Project.
4. Dra in mappen eller ladda upp innehållet.
5. Klicka Deploy.

När länken är klar:
- öppna den i Safari på iPhone
- tryck Dela
- välj Lägg till på hemskärmen

Notering:
Appen sparar data lokalt i webbläsaren.
Om två spelare använder olika mobiler behöver ni exportera/importera data mellan enheterna.
