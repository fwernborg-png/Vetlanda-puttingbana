KOD TILL INDEX.HTML

1) Lägg detta precis före </head>:

<link rel="manifest" href="/manifest.json">
<link rel="apple-touch-icon" href="/apple-touch-icon.png">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32.png">

<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-title" content="Vetlanda Puttingbana">
<meta name="application-name" content="Vetlanda Puttingbana">
<meta name="theme-color" content="#050805">


2) Lägg detta precis efter <body>:

<div id="splashScreen" class="splash-screen" aria-hidden="true"></div>


3) Lägg detta i <style>, gärna längst ner före </style>:

.splash-screen {
  position: fixed;
  inset: 0;
  z-index: 99999;
  background:
    linear-gradient(rgba(0,0,0,0.08), rgba(0,0,0,0.08)),
    url("/splash.png") center center / cover no-repeat;
  opacity: 1;
  visibility: visible;
  transition: opacity 0.55s ease, visibility 0.55s ease;
}

.splash-screen.hide {
  opacity: 0;
  visibility: hidden;
  pointer-events: none;
}

@media (min-width: 700px) {
  .splash-screen {
    background-size: contain;
    background-color: #050805;
  }
}


4) Lägg detta längst ner i befintliga <script>, precis före </script>:

setTimeout(() => {
  const splash = document.getElementById("splashScreen");
  if (splash) splash.classList.add("hide");
}, 2000);


5) Lägg filerna i huvudmappen:
- splash.png
- apple-touch-icon.png
- icon-192.png
- icon-512.png
- favicon-32.png
- manifest.json
